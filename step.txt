Server Side
    Créer le projet : 
    - npm init -y
    Installer les nodes_modules et les dépendances : 
    - npm install express mysql2 cors nodemon cookie-parser jsonwebtoken bcrypt
    Lancer le serveur : 
    - npm start
Front-end
    Créer le projet avec Vite : 
    - npm create vite@latest .
    Installer les nodes_modules et les dépendances : 
    - npm install bootstrap axios react-router-dom
    Lancer le projet sur un navigateur web :
    - npm run dev
